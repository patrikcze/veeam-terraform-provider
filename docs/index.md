# Veeam Provider

The Veeam provider allows you to manage [Veeam Backup & Replication](https://www.veeam.com/vm-backup-recovery-replication-software.html) V13 resources using Terraform. It communicates with the Veeam REST API (port 9419 by default).

## Authentication

The provider authenticates via OAuth2 using username/password credentials. Credentials can be provided directly in the provider block or via environment variables.

### Environment Variables

| Variable | Description |
|---|---|
| `VEEAM_HOST` | Veeam server hostname or IP address |
| `VEEAM_PORT` | REST API port (default: `9419`) |
| `VEEAM_USERNAME` | Username (e.g. `DOMAIN\admin`) |
| `VEEAM_PASSWORD` | Password |
| `VEEAM_INSECURE` | Skip TLS verification (`true`/`false`, default: `false`) |

## Example Usage

```hcl
provider "veeam" {
  host     = "veeam.example.com"
  username = "administrator"
  password = var.veeam_password
  # port    = 9419   # optional, defaults to 9419
  # insecure = false # set to true only for self-signed certificates in lab environments
}
```

Or using environment variables only:

```hcl
provider "veeam" {}
```
